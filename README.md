# ESP8266 + Arduino Tools

Kumpulan contoh & utilitas untuk ESP8266 dan Arduino: LED blink, wifi scanner, dan network tools dengan logger & LED status.

## Projek
- `LED_Blink` — contoh dasar GPIO (LED)
- `Wifi_Scanner` — scan jaringan Wi-Fi sekitar, output serial/HTTP
- `NetworkTools` — ping/ARP/device discovery

## Cara pakai singkat
1. Pilih folder project yang ingin dicoba
2. Buka file `.ino` di Arduino IDE
3. Pilih board (NodeMCU / Wemos / Arduino)
4. Upload dan pantau Serial Monitor
